Guia de Instalação e Uso do Front18

O Front18 Security é o seu escudo avançado de proteção de conteúdo.

Este manual foi desenhado para te guiar passo a passo na sua configuração, de forma direta e sem complicações focando perfeitamente no que deve e no que não deve ser alterado.

1. Instalação Descomplicada no WordPress

A preparação do seu site leva menos de três minutos. Siga as instruções abaixo para ativar sua blindagem:

  1. Certifique-se de ter o arquivo front18-wp-plugin.zip salvo no seu computador.
  2. Acesse o painel administrativo do seu site (o menu wp-admin).
  3. No menu lateral, vá até Plugins > Adicionar Novo e clique em Enviar Plugin no topo da tela.
  4. Selecione o arquivo ZIP que está no seu computador e clique em Instalar Agora.
  5. Após o upload e finalização do carregamento, clique no botão para Ativar Plugin.

2. Ações Seguras e Recomendadas (Onde Mexer) ✅

Na barra lateral preta do seu WordPress, um novo menu chamado Front18 estará presente. É através deste painel que você orquestra as defesas principais do seu portal.

🛡️ Configuração Principal

Estas são as ações do dia a dia. Você deve e pode alterar estas chaves sempre que necessitar:

  • Status do Front18: Utilize o interruptor de ativação no topo da tela. Ao ligá-lo, o seu site fica preparado para receber as regras gerais de proteção contra cópia e scraping.
  • SaaS API Key / Client ID: Cole neste campo a credencial única que nossa equipe forneceu. Utilize o ícone de "olho" no canto direito para conferir visualmente se não colou espaços vazios acidentalmente.
  • Botão de Salvar: Nunca confirme sua configuração sem clicar no botão "Salvar Blindagem" localizado no final da página para firmar suas regras com a nuvem em tempo real.

⚡ Flexibilidade (Exceções de Página)

Embora a proteção Front18 abranja todo o site globalmente, entendemos que documentos públicos (Termos de Uso, Política de Privacidade) não podem ser completamente bloqueados.

  • Abra normalmente a tela de edição ou o construtor da página que deseja liberar.
  • Na aba de configurações da direita, deslize até encontrar o widget: 🛡️ Defesa Front18.
  • Selecione a opção 🟢 Forçar Acesso (Liberar Sempre) e atualize/publique a página. A partir deste momento essa página específica desativará o SDK de bloqueio.

3. Área de Perigo (Onde NÃO Mexer) ❌

Ao navegar para o final do painel de Configurações, há uma seção oculta nomeada "Painel de Desenvolvedor". Esta zona não deve ser alterada sem supervisão técnica.

⚠️ Integração Customizada Avançada

Aviso: A modificação de quaisquer dados presentes nos campos abaixo resultará na inoperância do plugin e, consequentemente, afetará o acesso do usuário final ao site.

  • URL Base do SDK Front18: Alterar este link desvincula o plugin da nossa infraestrutura em nuvem, fazendo com que as checagens percam a comunicação.
  • Objeto Padrão (JS): Alterações desreguladas gerarão erros na renderização e problemas de UI no computador do visitante.
  • Chave Interna de Token: Esta credencial comunica diretamente o seu servidor com a nossa API Core. Resetá-la causará instabilidade crítica.

* Exceção: A opção Telemetria Console só deve ser habilitada a pedido direto de um especialista do Suporte Nível 3 para sessões de depuração online.
